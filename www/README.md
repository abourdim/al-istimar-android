# 🏴 Colonialism: Grudges & Greed — الاستعمار أحقاد وأطماع

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الاستعمار أحقاد وأطماع (Colonialism: Grudges and Greed)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on colonial history, exploitation, resistance, cultural domination, independence movements
- 5 resistance solutions with problem/solution format
- Daily habit tracker with streaks, self-assessment quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes (Earth, Night, Resistance) with localStorage persistence
- Narrator with karaoke, voice/speed/pitch, sleep timer
- Responsive design, swipe navigation, sound effects, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
